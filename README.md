*  Описание:

*

Мой первый проект. Предварительное имя "2047".

Это веб версия моего проекта на Ren`Py.
В детстве я любил "Бесконечное лето", и, наконец, могу сделать игру в том же жанре.
Если кому-то здесь вообще интересен сюжет, то здесь он закручивается вокруг искуственного интеллекта и испытания мозговых чипов.
Футуристично, затрагивает актуальные проблемы развития ИИ и, конечно же, обилие анимешных красоток.
